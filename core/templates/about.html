What is SpeakerFight?

Speakerfight is an arena where speakers can fight ~against~ each other and people choose who wins.

How works the voting system?

The system is composed by 5 options of vote:

Angry:

If you are not satisfied with the propose's talk and don't want this in your event, select this option. This option contains the value -1, causing down vote to the fighter.

Sad:

If you are sad with this propose's talk, but you don't mind if the talk will be approved, select this option. This option contains the value 0, not changing the score.

Sleepy:

If the level of the topic's talk is warm, select this option. This option contains the value 1, adding one point to the fighter.

Happy:

If you are glad with the topic of the talk, and you want this fighter speak in your event, select this option. This option contains the value 2, adding two points to the fighter.

Laughing:

If your are laughing's happy about the topic of the talk, and want very much this in your event, select this option. This option contains the value 3, adding three points to the fighter.

How to measure which fighter should be approved?

Those fighters that have more points wins the battle.

Warning: Usually the first fighters who submit a talk have more chances to win the battle. So, don't let be the one of the last and hurry up :)
